# jiang-nan-skill

一个基于江南作品公开语料与本地整理资料蒸馏出的虚构陪伴 Skill。

它的目标是把江南作品中稳定出现的高层创作模式整理成一个可调用的聊天与写作伙伴：少年感、孤独、热血、错过、宏大系统、人物代价和故事余温。

## 适合做什么

- 像朋友一样聊故事、青春、孤独、热血和写作。
- 分析一个故事为什么“不够江南式地动人”。
- 设计原创青春幻想、都市奇幻、架空史诗或科幻爱情故事。
- 帮你把人物从“很强”改成“有弱点、有代价、有想守住的东西”。
- 把世界观从“名词堆砌”改成会压迫人物的系统。
- 给大纲、场景、人物关系做创作诊断。

## 不适合做什么

- 续写《龙族》《九州缥缈录》《上海堡垒》《此间的少年》等版权作品的官方剧情。
- 输出江南作品全集、长篇摘录或未授权文本。
- 编造江南的采访、出版事实、剧情和原话。

## 安装方式

### Claude Code / OpenClaw / QClaw

如果你的工具支持从 GitHub 安装 Skill，发布到 GitHub 后可以使用类似命令：

```bash
npx skills add <your-github-username>/jiang-nan-skill
```

也可以手动安装：

```bash
git clone https://github.com/<your-github-username>/jiang-nan-skill.git
```

然后按你的客户端要求，把仓库放到 Skill 搜索目录中。

### 手动复制

把整个 `jiang-nan-skill` 文件夹复制到你的 Skills 目录。常见路径示例：

```text
~/.claude/skills/jiang-nan-skill
~/.codex/skills/jiang-nan-skill
```

不同客户端的路径可能不同，以实际文档为准。

## 使用示例

聊天：

```text
使用 jiang-nan-skill。今晚有点孤独，陪我聊聊故事。
```

创作诊断：

```text
使用 jiang-nan-skill。这个主角是个普通高中生，被秘密学院录取，但我觉得他不够动人，请帮我重塑人物代价。
```

原创设定：

```text
使用 jiang-nan-skill。帮我设计一个受江南作品精神启发的原创都市奇幻故事，不要使用龙族、卡塞尔、路明非等既有设定。
```

场景改造：

```text
使用 jiang-nan-skill。把这个告别场景改得更有少年感和余味，但不要仿写江南原文。
```

研究分析：

```text
使用 jiang-nan-skill。分析《龙族》和《上海堡垒》里“错过”这个主题的差异。
```

## 设计原则

这个 Skill 抽取的是高层创作模式，不是句式模板。

核心公式：

```text
脆弱少年 + 庞大系统 + 短暂自由 + 关系见证 + 昂贵选择 + 事后余像
```

它会优先追问：

- 主角到底想守住什么？
- 代价是什么？
- 世界为什么不允许他轻易得到？
- 谁真正看见了他？
- 故事结束以后，什么东西还留在原地？

## 语料说明

本 Skill 的蒸馏参考来自本地整理的江南作品全集目录、长篇/短篇作品、散文随笔、文论、序跋、访谈录和设定集。仓库只保留抽象分析与创作规则，不随仓库发布原文语料。

如果你要做严格考据，请自行准备合法来源，并在回答前核对原文或正式出版资料。
