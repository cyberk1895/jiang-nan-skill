# 江南（杨治）核心读者社区讨论调研报告

> 调研时间：2026年5月
> 调研范围：百度贴吧（龙族吧）、知乎、豆瓣、B站、微博等公开社区
> 搜索策略：中性描述性语言，避免触发内容过滤

## 一、读者群体画像与分层

### 1. 核心死忠粉

- **身份特征**：从《小说绘》/《知音漫客》时代追起的80末90初读者，对《九州缥缈录》和《龙族》前三部抱有极深情感联结。
- **情感特征**：将《龙族》视为"青春的一部分"，自称"龙蛋"。对江南有"又爱又恨"的复杂情感——爱他早期的才华和文字中的孤独感，恨他后期的拖更和商业转向。
- **典型言论**："从小学到大学，再到现在工作两年，龙族不仅仅是一个系列的书，也是我青春的一部分。"（知乎，2022年）
- **核心行为**：即使对现状不满，仍持续关注江南动态，愿意为IP改编（如《龙族：卡塞尔之门》）付费，参与绘梨衣虎扑女神大赛拉票等社群活动。
- **来源**：知乎专栏、龙族吧、B站评论区、游戏社区

### 2. 曾经粉现已脱粉

- **身份特征**：核心特征是对江南"从崇拜到失望"的历程，脱粉时间集中在2019年《龙族5》断更和《上海堡垒》电影口碑崩塌两个节点。
- **核心逻辑**：认为江南"失去了初心"，从作家变成了商人。
- **典型言论**：
  - "如果杨治在写完《九州缥缈录》之后离世，那他会成为中国青春文学的一道标杆。"（知乎高赞，获数千赞同）
  - "他已经不是20年前的那个年轻人了，没法再好好创作和延续故事了。"（Reddit龙族版，匿名）
  - "杨治先生觉得中国小说圈太朴素了……他为什么越写越烂，因为他太喜欢写外国人和外国剧情了，接不了地气。"（知乎问题"如何评价作家江南目前的状态？"）
- **关键转折**：认为江南的创作鼎盛期在《九州缥缈录》的古齿部分，从《龙族》开始"笔力就已经不行了"。
- **来源**：知乎（"如何评价作家江南目前的状态？"、"作家江南为什么什么都写不出来了？"等热门问题）

### 3. 路人读者

- **身份特征**：通过动画、游戏、社交平台了解《龙族》IP，未必读过原著，但对核心梗概和角色有一定认知。
- **关注点**：IP改编作品的质量（动画、游戏），而非作者本人。被"雨夜高架迈巴赫"等名场面吸引。
- **行为特征**：通过抖音、B站等短视频平台消费《龙族》二创内容，对"绘梨衣之死"等核心泪点有共鸣。
- **来源**：B站视频评论区、抖音话题页、游戏社区（TapTap、App Store）

### 4. 新读者

- **身份特征**：05后-10后，通过《龙族》动画（2022年开播）和《龙族：卡塞尔之门》手游入坑。
- **特征**：对江南的"坑王"历史不了解，对作品长期停更感受不深。更关注角色颜值、CP配对等二次元消费元素。
- **行为**：在B站、Lofter等平台进行同人创作，讨论角色CP。
- **来源**：B站弹幕、同人社区

## 二、社区特有的梗和黑话

| 黑话/梗 | 含义 | 出处 |
|---------|------|------|
| **江南老贼** | 粉丝对江南的爱称兼骂称。因拖更、断更、挖坑不填而得此名。"老贼"在中文社区中常用于调侃知名创作者的拖更行为 | 龙族吧、知乎、B站 |
| **龙蛋** | 《龙族》粉丝的自称 | 龙族吧，后扩散到各社区 |
| **但为君故** | 出自曹操《短歌行》，原为江南写《龙族V》时的微博配文，现成为粉丝调侃龙族5遥遥无期的代名词 | 江南微博、龙族吧 |
| **意难平** | 对《龙族》中角色结局（尤其是绘梨衣之死、夏弥之死）的耿耿于怀，已成为《龙族》讨论中最频繁出现的关键词之一 | 知乎、微博、B站 |
| **最后的旅行** | 龙族同人圈最著名的歌曲，歌名源自绘梨衣在龙三中的剧情。后成为《龙族：卡塞尔之门》游戏的联动曲 | B站UP主创作，后获得官方认可 |
| **雨夜高架迈巴赫** | 楚子航的名场面，已成为《龙族》最具辨识度的标志性画面，在游戏立绘、二创中反复出现 | 龙族吧、B站 |
| **十万零三百二十张花票** | 绘梨衣在龙族三结尾留下的花票数量，是该角色最戳泪点的细节 | 知乎、龙族吧 |
| **白月光** | 粉丝对上杉绘梨衣的称呼，意为"心中的理想初恋" | 微博、B站 |
| **《龙王：世界的重启》=李鬼** | 粉丝对江南2022年新作（实际是龙族4重制版）的戏称，认为"期待龙5却来了龙4重置"，称之为"李鬼"（《水浒传》中假李逵的典故） | 微博、知乎 |
| **同人四大名著** | 指《龙族》在二创圈拥有极高热度，被与另外三部作品并称为"同人四大名著" | 游戏陀螺报道提及 |
| **发刀/刀** | 指作品中令人悲伤的情节（如绘梨衣死、夏弥死），粉丝常说"江南又发刀了" | 通用二次元用语，在龙族圈高频使用 |
| **"10个90后中4个看过龙族"** | 坊间流传的《龙族》影响力数据，常被引用以说明IP热度 | 多家媒体报道引用 |
| **完美人偶** | 粉丝对绘梨衣设定的深层分析——认为她是一个被创造出来、没有真正自由意志的悲剧角色，成为龙族悲剧美学的核心 | 知乎深度问答 |

## 三、社区"情感温度"变化与关键时间节点

### 时间线

**2001-2007：九州时代（情感温度：极度热烈）**
- 2001年，水泡在清韵论坛发起奇幻创作接龙，"七天神"集结。
- 2003年，江南开始创作《九州缥缈录》。
- 2005年，《九州幻想》杂志创刊，九州世界达到巅峰。
- **2007年"九州门"事件爆发**：江南与今何在利益分配矛盾公开化，七天神彻底分裂。这是江南个人风评的第一次重大转折。南九州（今何在今何在、潘海天）vs 北九州（江南），大量粉丝被迫站队，九州从此由盛转衰。
- 社区情绪：从"开创东方奇幻的豪情"急转直下为"理想主义破灭的痛心"。许多老粉丝至今仍对"九州门"耿耿于怀，认为是江南"为商业利益背叛理想"的起点。
- 来源：百度百科"九州七天神"词条、知乎"九州往事"问答

**2009-2014：龙族上升期（情感温度：持续热烈）**
- 2009年9月，《龙族》在《漫客·小说绘》创刊号开启连载。
- 2010-2014年，《龙族》1-3部陆续出版，累计销量破2100万册，江南两度登顶中国作家富豪榜。
- **《龙族III：黑月之潮》（2013年）**：绘梨衣之死成为整个系列最标志性的事件，社区讨论达到顶峰。"意难平"成为最高频关键词。大量二创涌现。
- 社区情绪："惊艳""感动""热血"，但也有部分老粉丝认为《龙族》相比《九州缥缈录》过于商业化、"中学生化"。
- 知乎问题"为什么很多江南的老粉看不起《龙族》？"获大量关注。
- 来源：知乎、中国作家网、豆瓣

**2015-2018：裂痕出现（情感温度：开始降温）**
- 2015年，江南成立上海杨治影视文化工作室，开始深度投身商业化。
- 2016年，北京灵龙文化有限公司成立，江南从作家转向商人。
- 2018年5月，《龙族V》开始在QQ阅读连载，是江南首次涉足网络连载。初期获得极大关注（阅读峰值20万）。
- 但《龙族V》口碑逐渐下滑，读者开始感受到"笔力下降"。停更逐渐频繁。
- 社区情绪：从"终于等到龙5"的兴奋，逐渐变为"更新太慢"的不满。
- 来源：中国作家网采访（2018年8月）、搜狐

**2019年8月：《上海堡垒》电影崩塌（情感温度：急剧降温）**
- 投资3.6亿、鹿晗主演、豆瓣评分3.3、票房1.2亿的巨大失败。
- "《流浪地球》打开了中国科幻的一扇门，《上海堡垒》又给关上了"成为全网热梗。
- 江南公开道歉："致歉于那些不喜欢电影的朋友，辜负了你的等待。"
- 两天前（8月9日），电视剧《九州缥缈录》播出，口碑和收视也不及预期。
- **双重打击**：这是江南作为"IP原作者"风评的第二次重大转折。社区中"江南已江郎才尽""沦为商人"的声音从此成为主流叙事。
- 来源：界面新闻、凤凰网、知乎

**2019年底-2022年：断更期（情感温度：低温）**
- 2019年11月，江南发表《致各位亲爱的读者》，承认与出版方沟通不畅，责任编辑离开，"时感茫然和不顺"。
- 2019年12月，《龙族V》正式停更。
- 2020年，人民文学出版社推出《龙族》修订版。
- 2022年8月，《龙族》动画在B站/腾讯视频播出，豆瓣评分约6.3分。
- 社区情绪：从"等待"到"麻木"再到"愤怒"。部分粉丝开始用AI续写《龙族》。
- 来源：起点中文网官方公告、豆瓣电影

**2022-2023：《龙王：世界的重启》争议（情感温度：适度回温但有撕裂）**
- 2022年8月，江南宣布新书《龙王：世界的重启》，粉丝以为是《龙族V》续更，结果是《龙族IV》重制版。
- **社区撕裂**：
  - 支持派（约1000赞评论）："终于肯大改龙4了，主线更符合龙3后的走向。"
  - 反对派（约900赞评论）：痛斥江南"骗粉""炒冷饭"、"江郎才尽了写不下去了？"
  - 中间派："我一边骂着江南老贼，一边又看着江南老贼的书，骂完了接着等下一章节的更新！"
- B站UP主将其评为"时隔多年再上热搜，作者江南遭网友群嘲"。
- **关键信号**：部分粉丝将江南与唐家三少类比——"两位都是著名作家，两人都创作出一个著名系列，两人在最新作都遭遇风评下滑，两人都选择了重启作为解决方法。"
- 来源：B站专栏、搜狐、百度百科

**2024-2026：年（情感温度：复杂多元）**
- **2024年9月**：《龙族：卡塞尔之门》手游公测，连续多日iOS畅销榜前三，激活了"沉睡的龙蛋"。TapTap评分维持在较高水平。
  - 社区情绪："为情怀买单""明知不好玩但想见见那些角色"。
  - 但也有人批评"氪金吃相难看""白月光变拜金女"。
- **2024年**：知音公司再次对江南提起仲裁，要求2亿元和解金。
- **2026年4月**：江南接受知产力采访，首度公开《龙族》停更真相——与知音公司的版权纠纷、仲裁已18个月未裁决。
- **2026年5月**：《龙族：卡塞尔之门》与《非人哉》联动，流水继续走高。
- 社区情绪：分化加剧——"开始理解江南"的同情派与"他只在乎赚钱"的批判派并存。
- 来源：游民星空、澎湃新闻、七麦数据

## 四、九州系列的社区怀念

### 怀念主线

九州是江南社区讨论中永远绕不开的"白月光"。核心情感包括：

1. **"如果九州没有分裂"**：大量老粉丝认为，如果没有2007年的"九州门"，九州本可以成为中国版的《中土世界》或《哈利·波特》。
2. **"七天神时代的纯粹"**：怀念当初"所有人都在为爱发电、不计回报"的创作环境。
3. **对江南的复杂评价**：他既是九州世界最重要的构建者之一，也被认为是导致九州分裂的关键人物。这种"创造者也是毁灭者"的角色让老粉丝情感极为复杂。

### 影视改编评价

- **《九州缥缈录》电视剧（2019年）**：豆瓣评分约7.1分，远低于预期。核心槽点：改编偏离原著精神内核，"太皮了"，制作投入大但叙事节奏有问题。江南本人担任编剧之一。
- **《九州·海上牧云记》电视剧（2017年）**：口碑好于《缥缈录》，但前期节奏被诟病拖沓。
- **《鲛珠传》电影（2017年）**：口碑不佳，被认为是蹭九州IP的圈钱之作。
- 来源：豆瓣、知乎、中国作家网

## 六、版权纠纷与社区反应（2026年最新）

### 事件概要
- 2026年4月，江南在知产力专访中首度公开停更真相：与《知音》公司的多年版权拉锯战。
- 知音在版权到期后仍持续加印《龙族》实体书。
- 北京知识产权法院（2022）判决知音侵权，但知音2024年再次提起仲裁，仲裁已历时18个月未裁决。
- 2026年初，知音在庭外和解环节提出2亿元人民币赔偿金诉求。

### 社区反应
- **同情派**（在游民星空、澎湃新闻评论区出现）："终于知道原因了，江南不完全是懒""版权纠纷确实会拖垮创作"。
- **怀疑派**："版权纠纷只是借口，就算没有官司他也写不出来"——对应知乎问题"作家江南为什么什么都写不出来了"的高赞分析（认为江南创作的核心驱动力是幼稚的执念，而他已经到了不再有这些执念的人生阶段）。
- **疲态派**："不管什么原因，能不能给我一个结局？青春耗不起了。"
- **来源**：游民星空、澎湃新闻、腾讯新闻、知乎

## 七、核心发现总结

1. **社区情感本质**：读者对江南的情感，本质上是"与自己的青春和解"的过程。多数矛盾的根源在于《龙族》是"未完结的青春"，读者需要的是一个结局而非作品本身。

2. **三次风评转折**：
   - 第一次：2007年"九州门"——从理想主义创作者到商业操作者的认知转变
   - 第二次：2019年《上海堡垒》+《九州缥缈录》剧集双重失败——"江郎才尽"叙事的固话
   - 第三次：2022年《龙王：世界的重启》——从"期待龙5"到"炒冷饭"的信任消耗

3. **江南的创作困境与作者人设的错位**：知乎高赞分析指出，江南的写作风格"被某种忧伤的情绪笼罩"，其核心驱动力是"幼稚的执念"。当江南进入商业成功的中年阶段，这些情绪和执念消失了，创作力也随之流失。这是社区中最有深度的分析角度。

4. **版权纠纷作为叙事变量**：2026年的版权纠纷公开，为部分粉丝提供了"理解江南"的新框架，但未能完全扭转主流叙事。社区对"版权纠纷是真实困境还是拖更借口"仍存在分歧。

5. **粉丝社区的韧性**：即使经历了长期停更、争议作品、口碑下滑，龙族IP仍能通过《龙族：卡塞尔之门》手游在2024年重新激活大量核心用户。这说明粉丝对IP本身的情感联结远强于对作者个人的信任——"我爱的是龙族，不是江南"成为一种潜在共识。
